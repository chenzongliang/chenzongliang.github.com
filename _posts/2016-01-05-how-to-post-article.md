## 编写文章
首先在Windows下使用MarkdownPad工具编辑文章，MarkdownPad下载地址：[点击链接下载](http://markdownpad.com/)

## 使用Git工具提交文件
添加当前所有修改文件
    
    git add .

提交当前文件到本地仓库

    git commit -a -m 'add files'

提交本地仓库代码到远端git服务器

    git push origin master
